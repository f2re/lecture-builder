## Чеклист рецензии

### 1. Соответствие ФГОС и компетентностному подходу
- [ ] Тема соответствует рабочей программе дисциплины
- [ ] Все заявленные компетенции (ОК, ОПК, ПК) раскрыты в содержании
- [ ] Присутствуют все уровни по Блуму: знание → понимание → применение
- [ ] Формулировки целей «Знать/Уметь/Владеть» соответствуют ФГОС 3++

### 2. Соответствие теме и вопросам
- [ ] Каждый учебный вопрос полностью раскрыт
- [ ] Нет отклонений от заявленной темы
- [ ] Объём каждого раздела соразмерен значимости вопроса
- [ ] Введение и заключение соответствуют содержанию

### 3. Научность и достоверность
- [ ] Все утверждения подкреплены ссылками
- [ ] Используются актуальные источники (не старше 5-7 лет для основных)
- [ ] Определения соответствуют принятым в данной дисциплине
- [ ] Нет фактических ошибок

### 4. Формулы и математический аппарат
- [ ] Все формулы записаны в LaTeX-нотации
- [ ] Все символы в формулах расшифрованы
- [ ] Нумерация формул последовательна
- [ ] Ссылки на формулы в тексте корректны
- [ ] Вывод формул (если есть) логически верен

### 5. Логика и структура изложения
- [ ] Прослеживается логическая цепочка от простого к сложному
- [ ] Переходы между разделами обоснованы
- [ ] Нет смысловых повторов между разделами
- [ ] Выводы вытекают из содержания

### 6. Педагогическая доступность
- [ ] Уровень сложности соответствует курсу и специальности
- [ ] Термины объясняются при первом употреблении
- [ ] Есть примеры и иллюстрации к абстрактным понятиям
- [ ] Контрольные вопросы проверяют все ключевые элементы

### 7. Оформление библиографии
- [ ] Все источники оформлены по ГОСТ Р 7.0.5-2008
- [ ] Разделение на основную и дополнительную литературу
- [ ] Все источники из текста присутствуют в списке
