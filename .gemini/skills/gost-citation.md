# Skill: GOST R 7.0.5-2008 Citation Format

Standard templates for Russian academic bibliography citations.

## Book (textbook / monograph)
```
Фамилия И.О. Название: подзаголовок / И.О. Фамилия. — Город: Издательство, Год. — N с. — ISBN ...
```
Example:
```
Хромов С.П. Метеорология и климатология: учебник / С.П. Хромов, М.А. Петросянц. — М.: Изд-во МГУ, 2012. — 582 с.
```

## Journal article
```
Фамилия И.О. Название статьи / И.О. Фамилия, И.О. Соавтор // Название журнала. — Год. — Т. N, № M. — С. X–Y. — DOI: ...
```

## Web resource
```
Фамилия И.О. Название страницы [Электронный ресурс] / И.О. Фамилия // Название сайта. — Год. — URL: https://... (дата обращения: ДД.ММ.ГГГГ).
```

## Normative document
```
ГОСТ Р 7.0.5-2008. Библиографическая ссылка. — М.: Стандартинформ, 2008. — 23 с.
```

## Rules
- Up to 3 authors listed; 4+ → first author + `[и др.]`
- Year always 4 digits
- Russian city abbreviations: **М.** (Москва), **СПб.** (Санкт-Петербург), **Л.** (Ленинград)
- Access date required for all web resources
- DOI mandatory when available
