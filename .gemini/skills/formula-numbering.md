# Нумерация формул по ГОСТ

## Общие правила оформления формул

Формулы в лекциях должны оформляться в соответствии с **ГОСТ Р 7.0.11-2011** и **ГОСТ 2.105-95**.

### Размещение номера формулы
- Номер формулы указывается **арабскими цифрами** в **круглых скобках**
- Номер размещается **справа от формулы** на той же строке, в крайней правой позиции
- После номера формулы точка **не ставится**

### Система нумерации в лекциях
Используется **двухуровневая нумерация** в формате `(N.M)`, где:
- **N** — номер лекции
- **M** — порядковый номер формулы в данной лекции

**Примеры:**
- Лекция 1: (1.1), (1.2), (1.3), ...
- Лекция 5: (5.1), (5.2), (5.3), ...
- Лекция 12: (12.1), (12.2), (12.3), ...

### Типы формул в Markdown

#### Инлайн-формулы (без номера)
Используются для простых выражений внутри текста:
```
Согласно второму закону Ньютона \( F = ma \), сила прямо пропорциональна ускорению.
```

#### Блочные формулы (с номером)
Для важных формул, на которые будут ссылки в тексте:
```
\[ F = ma \] (5.1)
```

**Важно:** Номер формулы выносится **за пределы блока** `\[ \]` и размещается на той же строке справа в круглых скобках.

### Оформление в Markdown для конвертации в DOCX

#### Правильный формат для блочной формулы с номером:
```markdown
\[ F = ma \] (5.1)
```
или с выравниванием:
```markdown
\[ F = ma \] <span style="float:right">(5.1)</span>
```

#### Расшифровка символов
После каждой пронумерованной формулы необходимо расшифровать все символы:
```markdown
\[ F = ma \] (5.1)

где:
- \( F \) — сила, Н;
- \( m \) — масса тела, кг;
- \( a \) — ускорение, м/с².
```

### Ссылки на формулы в тексте
При ссылке на формулу в тексте указывается её номер в круглых скобках:
```markdown
Из выражения (5.1) следует, что при увеличении массы...
Подставляя значения в формулу (5.2), получаем...
Согласно уравнению (5.3), зависимость имеет линейный характер.
```

### Группы формул
Если несколько связанных формул объединяются в одну группу, используется единый номер:
```markdown
\[
\begin{cases}
x = r \cos\theta \\
y = r \sin\theta
\end{cases}
\] (5.4)
```

### Длинные формулы
Если формула не умещается на одной строке, её переносят на следующую строку по знаку операции. Номер ставится на последней строке:
```markdown
\[ E = m c^2 + \frac{1}{2} m v^2 + 
    \frac{G M m}{r} \] (5.5)
```

## Правила конвертации в DOCX

При конвертации Markdown → DOCX (через Pandoc):
1. LaTeX-формулы (`\[ \]` и `\( \)`) автоматически конвертируются в формат Office Math
2. Номера формул в круглых скобках справа сохраняются как текст
3. Для лучшего выравнивания в DOCX рекомендуется использовать таблицу:

```markdown
| Формула | Номер |
|:--------|------:|
| \[ F = ma \] | (5.1) |
```

Но в большинстве случаев достаточно простого размещения номера после формулы на той же строке.

## Контрольный чек-лист

При написании раздела лекции убедись, что:
- [ ] Все важные формулы пронумерованы в формате `(N.M)`
- [ ] Номера идут по порядку без пропусков: (5.1), (5.2), (5.3), ...
- [ ] После каждой пронумерованной формулы есть расшифровка символов
- [ ] В тексте есть ссылки на ключевые формулы
- [ ] Инлайн-формулы используются для простых выражений без необходимости ссылок
- [ ] Номер формулы размещён справа в круглых скобках
