# Skill: Нумерация формул (ГОСТ 2.105-2019)

## Стандарт
ГОСТ 2.105-2019 «ЕСКД. Общие требования к текстовым документам», п. 4.3.3.

## Формат номера
`(N.M)` — N = номер лекции, M = порядковый номер формулы в лекции.  
Примеры: (5.1), (5.2), (10.3)

## Правила
1. Номер ставится **справа** от формулы в круглых скобках — `(N.M)`
2. Нумерация сбрасывается в 1 при переходе к новой лекции
3. Нумеруются только **блочные** формулы; инлайновые — без номеров
4. `eq_counter` — агент ведёт счётчик, инкрементирует на каждую блочную формулу
5. Номер лекции (N) берётся из поля `lecture_number` YAML-конфига

## Markdown-формат (pandoc → DOCX)

### Блочная формула с номером
Используй `\tag{N.M}` — стандарт LaTeX, правильно конвертируется pandoc в OMML (Word):

```
$$
\rho = \frac{m}{V} \tag{5.1}
$$
```

Рендер: формула по центру, номер (5.1) справа — точно по ГОСТ.

### Инлайновая формула (без номера)
```
\( \rho = m/V \)
```

## eq_counter — алгоритм
```
eq_counter = 1  # сбрасывается для каждой новой лекции

для каждой блочной формулы:
    label = f"({lecture_number}.{eq_counter})"
    eq_counter += 1
```

После каждой блочной формулы обязательно:
> где ρ — плотность, кг/м³; m — масса, кг; V — объём, м³.
