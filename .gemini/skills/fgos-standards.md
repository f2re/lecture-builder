# Стандарты ФГОС 3++ для лекционных материалов

## Основные требования
- **Компетентностный подход**: Каждая лекция должна способствовать формированию конкретных компетенций (ОК, ОПК, ПК).
- **Результаты обучения**: Должны быть четко сформулированы через «Знать», «Уметь», «Владеть».
- **Актуальность**: Использование современных источников (не старше 5 лет для динамичных дисциплин).
- **Методическая обработка**: Текст должен быть адаптирован для самостоятельного изучения студентами.

## Формулировки целей
- **Знать**: фундаментальные основы, понятийный аппарат, классификации.
- **Уметь**: применять теорию к решению практических задач, анализировать данные.
- **Владеть**: навыками использования инструментов, методик, профессиональной терминологией.
